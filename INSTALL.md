# 開発環境インストールマニュアル

## Node.js

- [Node.js公式サイト](http://nodejs.org/ja/)
- [nodebrew](https://github.com/hokaccha/nodebrew) Node.js version manager.

## Node.jsモジュール

### グローバルモジュール

- gulp
- typings
- yarn （任意）

```sh
$ npm install -g gulp typings yarn
```

### ローカルモジュール

```sh
$ yarn
# or
$ npm install
```
